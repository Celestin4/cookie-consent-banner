This is a [Next.js] Cookie Consent Banner

## Getting Started

First, run the Production server on this Link : [Production-Link] (https://cookie-consent-banner.vercel.app/)

OR 


Second, run the development server:

```bash
git clone https://github.com/Celestin4/cookie-consent-banner.git
npm install
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


### Functionality
- Banner appears on first visit
- Saving the user selection
- Correct persistence of settings (rejected or accepted)
- Banner disappears after selection
